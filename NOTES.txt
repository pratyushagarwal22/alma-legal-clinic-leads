Attribution Note
================

The entire codebase was generated by the Cursor agent, task by task, against a
design doc (docs/design.md) and a dependency-ordered plan (docs/plan.md).

The author steered the process throughout: owning the architecture, scope,
layering rules, verification of each task, and all git commits. The agent wrote
code to spec; the author made the decisions and confirmed the results.

Each task was built in its own session and committed after manual verification.
Commit messages note which work was agent-generated versus hand-reviewed.
